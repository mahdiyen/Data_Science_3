1- use pipeline for normalization, PCA, and t_SNE , Then clustering

2- use pipeline for clustering without t-SNE

CONCLUSION:¶

I first loaded the data. Then using pipeline preprocessed data.I had 2 approaches:

1- use pipeline for normalization, PCA, and t_SNE , Then clustering

2- use pipeline without t-SNE

I run both approaches, and visualize them. As the plots shows, when we use 2 dimension reduction methods(PCA and t-SNE), and then cluster the data by K-means, it gives use better results than using one dimension reduction method.

However, by evaluating the methods based on silhouette matrix, it shows that without t-SNE the accuracy is better.

Explanation and conclusion:¶

GridSearchCV is a class in scikit-learn that performs an exhaustive search over a specified parameter grid to find the best combination of hyperparameters for a given estimator.

I used this class to find the best combination Kmeans parameters. The results shows that:

The best k-mean clustering is with 'kmeansinit': 'k-means++', 'kmeansn_clusters': 6, 'kmeans__random_state': 42.
